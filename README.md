# نِبْرَاس (Nibras) 🌙
**التاريخ الإسلامي كما لم تره من قبل.**

![Nibras Banner](https://images.unsplash.com/photo-1584286595398-a59f21d313f5?auto=format&fit=crop&q=80&w=2000&h=600)

## 📌 الفلسفة وجوهر المشروع
التاريخ ليس مجرد نصوص تُقرأ، بل هو **"مكان"** شهد الحدث، و**"زمان"** وثق اللحظة. "نبراس" هو المصباح الذي يضيء لك الزمان والمكان معاً، لتعيش السيرة النبوية كأنك تراها. 

---

## 🌟 الرؤية والرسالة
**رؤيتنا:** أن نكون المرجع المكاني التفاعلي الأول للمسلم المُعاصر، لاستكشاف السيرة النبوية والتاريخ الإسلامي بأسلوب بصري وتقني يربط القلوب بمواطن الوحي.

**رسالتنا:** توظيف أحدث تقنيات الخرائط التفاعلية (Web Mapping) لتوثيق أحداث السيرة النبوية وتاريخ الخلفاء الراشدين، وتقديمها خالية من الحدود السياسية الحديثة، بالاعتماد على أصح المصادر السنية المعتمدة.

---

## ✨ المميزات الرئيسية وإمكانيات التطبيق

### 📍 الزمان والمكان في نقطة واحدة
لا تقرأ التاريخ مجرداً. تتبع الأحداث عبر خط زمني (Timeline) تفاعلي يتزامن تلقائياً مع المواقع الجغرافية الحقيقية لغزوات ورحلات النبي ﷺ والصحابة.

### 🗺️ جغرافيا تحاكي الماضي
استمتع بخريطة مصممة خصيصاً لتعكس الواقع الجغرافي للقرن السابع الميلادي، بلا حدود سياسية حديثة، لتعيش الحدث في إطاره المكاني الأصلي.

### 📚 توثيق سني معتمد ومحتوى ديني غني
كل معلم على الخريطة مدعوم بملخص شامل يحتوي على:
- **تسلسل دقيق للأحداث** موثق من أمهات كتب السيرة (أهل السنة والجماعة).
- **آيات قرآنية** نزلت في سياق الحدث.
- **أحاديث نبوية دالة** من الصحاح.
- **تراجم وأدوار الصحابة** رضي الله عنهم بخاصية البحث السريع وعرض بطاقات تعريفية مفصلة.
- روابط بحث سريعة في **المكتبة الشاملة** للتحقق من المصادر.

### 🎬 السرد التفاعلي (رحلة عبر الزمن)
إمكانية تشغيل "جولة تفاعلية" تقوم آلياً بالتنقل بين المواقع الجغرافية وسرد تفاصيل الأحداث والتوسع في عرض الامتداد الجغرافي للدولة الإسلامية بشكل تسلسلي.

## 💻 المحتوى التقني (Technical Stack)

تم بناء **نِبْرَاس** باستخدام أحدث تقنيات الويب لضمان أداء عالٍ وتجربة مستخدم سلسة:

- **إطار العمل (Framework):** React 18 مع Vite (باستخدام TypeScript).
- **الخرائط التفاعلية (Maps):** مكتبة `Leaflet` و `react-leaflet` مع طبقات خرائط مخصصة تحاكي التضاريس التاريخية بدون حدود (Satellite & Terrain).
- **التصميم والواجهات (Styling):** `Tailwind CSS` مع تطبيق دقيق جداً للهوية البصرية الإسلامية والمخطوطات العتيقة.
- **الحركة الجمالية (Animations):** `Framer Motion` (أو `motion/react`) لتنقلات الشاشات وتأثيرات الخط الزمني بسلاسة.
- **الأيقونات:** `lucide-react` بتخصيصات لونية دقيقة.

### 🎨 الهوية البصرية المُضمنة برمجياً
- **خلفية المخطوطات (Parchment):** `#Fdfbf7` أو `#F5F5DC` - يعطي إحساس الورق القديم.
- **الأخضر الزيتوني (Olive Green):** `#4A5D23` - يرمز للطبيعة النبوية والقباب.
- **الذهبي النحاسي (Brass Gold):** `#B8860B` - للتأكيد وخطوط سير الغزوات.
- **حبر المخطوطات (Dark Ink):** `#1A1A1A` - لنصوص مريحة للعين.
- **الخطوط (Typography):** 
  - `Amiri`: للعناوين لتعكس الطابع الإسلامي والتاريخي.
  - `Tajawal` / `Cairo`: للنصوص لسهولة القراءة على الشاشات الحديثة.

## 🚀 تشغيل المشروع محلياً

1. قُم بنسخ المستودع:
   ```bash
   git clone https://github.com/your-username/nibras.git
   ```
2. ادخل لمجلد المشروع:
   ```bash
   cd nibras
   ```
3. قُم بتثبيت الاعتماديات:
   ```bash
   npm install
   ```
4. تشغيل خادم التطوير:
   ```bash
   npm run dev
   ```
